🧩 Provider Modes

mock

Safe deterministic local/CI/preview provider. Writes are dry-run.

official

Partial Meta Graph API boundary for supported account/profile/insight reads.

public

Read-only boundary for a compliant public-data upstream you control.

authorized

Disabled by default; live operations require reviewed consent-based integration.
